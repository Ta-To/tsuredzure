2021-08-24

Release file for http://security.debian.org/debian-security/dists/buster/updates/InRelease is not valid yet (invalid for another ~). Updates for this repository will not be applied.
  • docker build 時に apt update で発生
  • マシンの時刻がずれるのが原因とのこと
  • => WSL2環境: sudo hwclock -s が早い。ntpdate はデフォルトでは入っていない
  • / hwclock の hw はハードウェアのこと。マザーボード記録の時刻を参照できる
  • / WSL2環境で時刻がずれる問題はそのうち修正が入るらしい
  • / docker build がこけたときに発生するコンテナとイメージを一括で削除
    • docker rm $(docker ps -a --filter 'exited=100' -q) && docker rmi $(docker images --filter 'dangling=true' -aq)
    • 上記のフィルタ(--filter)については要確認
