  • docker elixir環境
  • cmark hex をインストールして使おうとしてのエラー

結論

  • 依存ライブラリをインストールする。apt install musl-dev
  • 見つからないのでリンクする。ln -s /lib/ld-musl-x86_64.so.1 /lib/libc.musl-x86_64.so.1
