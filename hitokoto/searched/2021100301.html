docker-compose 使用時にユーザのuid/gidを設定するには？

  • docker run ~ -u $(id -u):$(id -g) ~ 相当のことをしたい

下記の通り

  • 環境変数を通して指定する
  • 設定は user: "${USER_ID}:${GROUP_ID}" としておく
  • 起動時に USER_ID=$(id -u) GROUP_ID=$(id -g) docker-compose up -d 等と指定する
